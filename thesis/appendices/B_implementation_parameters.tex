\chapter{Controller and Firmware Parameters}
\label{app:implementation_parameters}

This appendix records the numeric configuration associated with the pinned
experimental dataset. Runtime tunables and sign settings were checked against
the session event logs. Compile-time timing, limit, and safeguard values were
checked against the repository firmware source. The logs do not embed a
firmware commit identifier, so those compile-time values are a reconstructed
implementation record rather than a cryptographically versioned historical
snapshot. The main text presents only the settings needed to explain each
controller. A dash denotes a dimensionless setting, not an unavailable value.
