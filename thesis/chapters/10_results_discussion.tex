\chapter{Results and Discussion}
\label{ch:results}

This chapter compares the pinned thesis dataset using the figures and tables generated by the analysis pipeline described in Chapter~\ref{ch:experiments}. Because each condition is represented by one pinned run only, all conclusions in this chapter are qualitative and metric-based.

\section{How to read this chapter}
The dataset used in this report consists of seven pinned representative trials: H1--H3 (hold), N1 (nudge), and T1--T3 (tap). The main body uses shared comparison plots so the three controllers can be read on the same scale. Supporting diagnostics are shown afterward for readers who want more detail.

\section{Upright hold-at-center (H1--H3)}
\noindent\textbf{Objective.} Each controller is engaged with $\theta_{\mathrm{ref}} \equiv 0$ and asked to balance upright without drifting toward the mechanical base limits.

\noindent\textbf{Key takeaways.}
\begin{itemize}
  \item The linear controller gives the lowest RMS upright error in the pinned hold runs.
  \item The hybrid SMC run gives the smallest peak $|\alpha|$ and the smallest peak $|\theta|$ in this fixed dataset.
  \item The full-surface SMC run also balances upright, but it uses much higher effort and shows noticeable centering drift in its representative trial.
\end{itemize}

\section{Commanded reference tracking / nudge mode (N1)}
\noindent\textbf{Objective.} Demonstrate balance-with-motion behavior by commanding a sequence of base targets $\theta_{\mathrm{ref}}(t)$ while maintaining upright balance. In the pinned dataset, only the linear controller is evaluated in nudge mode.

\noindent\textbf{Key takeaways.}
\begin{itemize}
  \item The pinned run follows the commanded sequence and keeps the pendulum upright during the full motion.
  \item The reversal steps show finite rise times of about \SIrange{3.1}{3.3}{s} and visible overshoot.
  \item No commanded step meets the fixed $\pm\SI{1}{\degree}$ settling criterion within the available post-step window, so the nudge result should be read as a successful motion demonstration rather than a tightly settled servo result.
\end{itemize}

\section{Finger tap disturbance (T1--T3)}
\noindent\textbf{Objective.} While balancing upright, the pendulum is disturbed by brief manual taps. The goal is to recover without falling or hitting base limits. These runs do not contain explicit disturbance markers, so the comparison here is qualitative.

\noindent\textbf{Key takeaways.}
\begin{itemize}
  \item The linear controller remains upright for the full pinned tap run.
  \item The hybrid SMC and full-surface SMC runs both end with a fallen outcome in this fixed dataset.
  \item In these representative runs, the hybrid SMC shows the largest peak angle excursion and the largest peak acceleration command, while the full-surface SMC shows the largest RMS acceleration effort.
\end{itemize}

\noindent\textbf{Interpretation.} The tap runs show the practical limit of this stepper-driven platform more clearly than the hold runs. When the controller asks for aggressive recovery, the measured state can say ``recover harder'' even when the open-loop stepper is already close to what it can realize. On this rig, that mismatch can keep the recovery in a high-effort region until the pendulum leaves the upright window or the base approaches a limit.

\section{Overall discussion and recommendations}
Based on this rig and this fixed dataset:
\begin{itemize}
  \item For calm steady hold-at-center behavior with low effort, the linear controller is the most consistent choice.
  \item For better peak suppression in the pinned hold run, the hybrid SMC is the strongest of the three, but its tap run shows that aggressive recovery can still fail on this platform.
  \item The full-surface SMC provides one unified nonlinear structure, but on this stepper-driven rig it is the most sensitive to actuator limits and protection logic.
  \item Nudge mode shows that commanded base motion while balancing is possible with the linear controller, but the pinned run does not show tight settling inside the fixed $\pm\SI{1}{\degree}$ band.
\end{itemize}
Because each condition is represented by one pinned run only, these conclusions should be read as qualitative observations, not as statistical claims.
