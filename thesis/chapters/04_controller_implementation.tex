\section{Common Implementation Boundary}
All three controllers use the same calibrated angles, derivative estimates,
reference signals, sign mapping, acceleration integration, stepper command
path, travel limits, and trial logging. This common boundary is important for
the comparison: differences in Chapter~\ref{ch:results} arise from the
controller and its mode-specific safeguards, not from different sensors or
data-acquisition paths.

The complete numeric parameter inventory used by the pinned trials is given in
Appendix~\ref{app:implementation_parameters}.
